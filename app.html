REWING: A CODING PROJECT

Here is a javascript coding exercise using arrays. The program displays your strings in reverse.

Note that your string(s) can be a Palindrome

"A palindrome is a word, number, phrase, or other sequence of characters which ca be read the same backward as forward, such as level, refer, radar."

Note that this application take care of perfect palindrome, i,e word by word, and without punctuation marks.
