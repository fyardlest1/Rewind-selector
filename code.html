Rewind Selector!

The Rewind Selector app allows the user to display strings in reverse".

Let's break this code out!

getValue()

The application is divided into three functions and starts with our controller function "getValue". This function takes into account an alert which will be displayed on the page in order to return the result of reverse string by calling two other functions "reverseString" and "displayReverseString" but also returns an alert when the user presses the button without entering any value.

reverseString()

Here we have created a logic function and it takes the value that will be passed by the user as parameter. This function defines the logic of the application, we started by creating an empty array and start with a "for loop" which takes into account the length of the text entered by the user, and starts an iteration from the end of the text to the beginning in order to rearrange the text from right to left, and add each new string in the "reverseString" variable.

displayReverseString()

The "displayReverseString" is our view function and it was created to display the alert messages into the interface and it takes two parameters: the value passed by the user and the string that was reverse.

That will allow the two values to be compared to see if they are identical. If the two values are the same the application will display two alert messages, the first will display the result of reverse String and the second will display an alert to tell you that your string is a palindrome.
